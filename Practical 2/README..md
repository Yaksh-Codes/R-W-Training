# 📊 Data Analyzer and Transformer Program

## Project Description

The Data Analyzer and Transformer Program is a Python console application that performs multiple operations on a dataset entered by the user.

The project demonstrates important Python concepts including:

- Functions
- Built-in Functions
- Lambda Functions
- Recursion
- Sorting
- Return Multiple Values
- Exception Handling

---

## Features

✔ Input Dataset

✔ Display Dataset Summary

✔ Calculate Factorial using Recursion

✔ Filter Data using Lambda Function

✔ Sort Dataset (Ascending / Descending)

✔ Display Dataset Statistics

✔ User-Friendly Menu System

## How to Run

Clone the repository

```bash
git clone https://github.com/YourUsername/Data-Analyzer-and-Transformer.git
```

Go into the project folder

```bash
cd Data-Analyzer-and-Transformer
```

Run the program

```bash
python source/data_analyzer.py
```
