# 📝 Interactive Personal Data Collector

A beginner-friendly Python project that collects personal information from the user and demonstrates:

- User Input
- Data Types
- Type Conversion
- Memory Addresses using `id()`
- Birth Year Calculation
- Formatted Output

---

---

## 🚀 Features

✅ Collect user's name

✅ Collect age

✅ Collect height

✅ Collect favourite number

✅ Calculate approximate birth year

✅ Display Python data types

✅ Display object memory addresses

---

## 🛠 Technologies Used

- Python 3.x
- datetime module

---


---

## ▶️ How to Run

Clone the repository

```bash
git clone https://github.com/yourusername/interactive-personal-data-collector.git
```

Go inside the folder

```bash
cd interactive-personal-data-collector
```

Run the program

```bash
python main.py
```
